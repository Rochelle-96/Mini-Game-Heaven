<h6> 숫자야구 <br>
Ordered 
1. 컴퓨터는 0~9까지 임의의 세자리 수를 정한다.<br>
2. 맞추는 사람이 임의의 세자리 수를 입력한다.<br>
3. 컴퓨터가 맞추는 사람이 입력한 세자리 수를 판독하여 스트라이크(S), 볼(B)의 개수를 알려준다.<br>
    1. 스트라이크(S)는 맞추는 사람이 부르는 세자리 수 중에서 정답의 같은 자리와 수까지 같은 경우이다.<br>
    2. 볼(B)은 맞추는 사람이 입력한 세자리 수 중에서 정답에 숫자만 있는 경우이다.<br>
    3. 아웃(O)은 1볼(B), 1스트라이크(S)도 없는 경우이다.<br>
4. 3스트라이크(3S, 임의의 숫자 세자리 수를 맞춘 경우)일 때 게임이 끝난다.<br><br>
※ 임의의 숫자를 적을 때 모두 다른 숫자로 적어야한다.<br>
ex) 554금지 (5가중복됨) </H6>
